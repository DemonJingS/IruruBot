



BOT的使用教程：
群里找到 jing_bot
私聊bot 发送 /start 命令 
bot会回复你一个带链接的按钮，点击即可使用。
bot有两个游戏模式：（单位：分钟）
1、游戏模式：这个结束时间需要其他玩家帮你加时才能有结束时间（四个难度）
简单（10-20） 普通（30-60） 困难（90-240）地狱（480-720）
2、倒计时模式：这个根据你输入的时间来决定你的结束时间（默认普通难度）

群聊使用方法：
发送/seelock可以看到自己的带锁信息，同时别人也能给你加时间
回复别人/seelock可以看到他人的带锁信息，同时可以给别人加时间
发送/vote 是投票解锁。
